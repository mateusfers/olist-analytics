Olist Analytics - Dashboard de Performance de Vendas

**Contexto do Projeto:**
Este projeto é um elemento de um plano de estudo focado em Análise de Dados.

**Objetivo:** Criar um dashboard interativo para analisar a performance de vendas do marketplace Olist.

## 🎯 Perguntas de Negócio
- Quais categorias de produto mais vendem?
- Qual o tempo médio de entrega e como isso impacta a nota do cliente?
- Qual a sazonalidade das vendas?
- Quais estados têm melhor performance?
